# Bookstore API

## Goal
Establish the following routes using the specified method:
- Users
    - CreateUser - POST
    - AuthenticateUser - POST
    - getAllUsers - GET
- Books
    - Create - POST
    - Delete - DELETE
    - LoanOut - POST
    - Return - POST
    - Update - PUT

## Tools
- Vanilla nodeJs(no framework)
- Jest
- Supertest
